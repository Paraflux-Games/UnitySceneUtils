# UnitySceneUtils
Utils package with scene helper functionality

Installation instructions:
* Install UniTask version 2.5.10
  * Open package manager in Unity, select "add package from git url" and put `https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask#2.5.10`
* Install UnitySceneUtils
  * Open package manager in Unity, select "add package from git url" and put `https://github.com/Paraflux-Games/UnitySceneUtils.git?path=Assets/Plugins/UnitySceneUtils`
